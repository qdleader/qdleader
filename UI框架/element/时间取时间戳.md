   
# 时间取时间戳

```js
   <el-date-picker
      v-model="value1"
      type="datetimerange"
      start-placeholder="开始日期"
      end-placeholder="结束日期"
      @change="changeDate"
      value-format="timestamp"
      :default-time="['12:00:00']"
    >
    </el-date-picker>

    用 value-format="timestamp" 就可以获取到时间戳格式的
```

时间组件制定某一默认时间

```js
<el-date-picker
   v-model="ruleForm.endValidTime"
   type="datetime"
   :default-time="new Date(2000, 1, 1, 23, 59, 59)"
   placeholder="请选择结束时间"
   style="width: 100%"
/>
```
