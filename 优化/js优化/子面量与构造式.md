# 子面量与构造式

// 构造式
```js
let test = () => {
    let obj = new Object();
    obj.name = 'yyy';
    obj.age = 18
    return obj
}
let str = new String('yyyy')
```

// 字面量
```js
let test = () => {
    let obj = {
        name: 'yyy',
        age: 18
    }
    return obj
}
let str = 'yyyy'

```

// 字面量 执行效率更快一些
