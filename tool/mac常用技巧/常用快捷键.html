常用快捷键

浏览器退出全屏

control + command + F

编辑器切换

control + 左右箭头

快速切换到桌面

除了可以用 command+F3 来显示桌面以外，我们还可以通过 Fn+F11 键来显示桌面。

打开我的电脑文件夹

Shift-Command-C：打开“电脑”窗口。
Shift-Command-D：打开“桌面”文件夹。

隔空投送

Shift-Command-R：打开“隔空投送”窗口

将所选项移到废纸篓

Command-Delete：将所选项移到废纸篓。
