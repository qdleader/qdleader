###组件的命名为多个单词

#组件名应该始终是多个单词的，根组件 App 以及 <transition>、<component> 之类的 Vue 内置组件除外。
这样避免和html元素冲突。



bad eg：

```
Vue.component('todo',{
	//...
})

export default {
	name:'ToDo'
	//..
}


```



good eg:

```
Vue.component('todo-item',{
	//..
})

export default {
	name:'TodoItem',
	//..
}
```


我认为驼峰方式 效果最好，因为大多数IDE自动完成功能都支持它。

bad

```
mycomponent.vue
myComponent.vue
Mycomponent.vue

```

good

```
MyComponent.vue
```
