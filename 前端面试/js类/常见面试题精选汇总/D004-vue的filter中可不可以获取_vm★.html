#и§Јжһҗпјҡ

жҜ”еҰӮеҪ“еҶҷдёҖдёӘvueж—¶й—ҙжҲіиҝҮж»ӨеҷЁж—¶еҖҷпјҢз”Ёthisи°ғз”ЁеҲ«зҡ„ж–№жі•:

execute_time(time) {
      if (time === 0) return '/'
      else return this.getTime(time)
    }

жҲ‘еҸ‘зҺ°жҠҘдәҶдёӘ"TypeError: Cannot read property вҖҳgetTimeвҖҷ of undefined"пјҢжҲ‘з¬¬дёҖж—¶й—ҙе°ұжү“еҚ°дәҶthisпјҢжһңз„¶жҳҜundefinedгҖӮ

е®ҳж–№ж–ҮжЎЈдёӯпјҡиҝҮж»ӨеҷЁfiltersзҡ„иҜҙжҳҺжҳҜпјҡ vueдёӯзҡ„иҝҮж»ӨеҷЁжӣҙеҒҸеҗ‘дәҺеҜ№ж–Үжң¬ж•°жҚ®зҡ„иҪ¬еҢ–пјҢ иҖҢдёҚиғҪдҫқиө–thisдёҠдёӢж–ҮпјҢеҰӮжһңйңҖиҰҒдҪҝз”ЁеҲ°дёҠдёӢж–ҮthisжҲ‘д»¬еә”иҜҘдҪҝз”Ёcomputedи®Ўз®—еұһжҖ§зҡ„жҲ–иҖ…дёҖдёӘmethodж–№жі•гҖӮ
