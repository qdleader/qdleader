
# B-005

```js
for(var index=0;index<5;index++){
  (function(){
    setTimeout(function(){
      console.log(index)
    },index*1000)
  })(index)
}
```

输出 		//5个5
			//for循环没有块级作用域 循环结束index为5 setTimeout为异步操作等循环结束后再执行

```js
for(var index=0;index<5;index++){
  (function(index){
    setTimeout(function(){
      console.log(index)
    },index*1000)
  })(index)
}
```

输出 0 1 2 3 4
