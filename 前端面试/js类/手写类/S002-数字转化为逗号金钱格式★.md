# 将数字12345678转化成RMB形式：12,345,678

　　思路：将字符串切割成数组再反转，遍历数组，加入辅助数组，当数组长度为3的倍数，再向辅助数组加入 ","。


```
 function Num(str) {
	 let arr = str.split('').reverse();
	 let res = [];
	 for(let i = 0; i < arr.length; i ++) {
		 res.push(arr[i])
		 if((i + 1) % 3 == 0) {
			 res.push(',')
		 }
	 }
	 return res.reverse().join('')
 }


```
