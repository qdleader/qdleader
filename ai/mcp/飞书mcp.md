# 飞书mcp

```js
{
  "mcpServers": {
    "feishu": {
      "command": "npx",
      "args": [
        "-y",
        "@larksuiteoapi/lark-mcp",
        "mcp",
        "-a",
        "你的飞书app_id",
        "-s",
        "你的飞书app_secret",
        "-u",
        "你的user_token"
      ]
    }
  }
}


```
