# Mastergo MCP

```js
{
  "mcpServers": {
    "mastergo-magic-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "@mastergo/magic-mcp",
        "--token=<MG_MCP_TOKEN>",
        "--url=https://mastergo.com"
      ],
      "env": {}
    }
  }
}
```

gitHub: https://github.com/mastergo-design/mastergo-magic-mcp?tab=readme-ov-file


获取 MG_MCP_TOKEN 方法：

访问 https://mastergo.com。

进入个人设置。

单击安全设置选项卡。

找到个人访问令牌，单击生成令牌。
